BUILD:
===================
falls erstmalig auf einem anderen Rechner installiert ODER configuration.ac geändert wurde: 
> autoreconf    // macht aus configure.ac neues configure und erstellt die Makefile.in 


Primal+Adjoint:
----------------
in SU2/:
clean distribution, if configure has been executed before. 
>make distclean

in SU2/:
>./preconfigure.py --prefix=$PWD --enable-mpi --with-cxx=mpicxx --with-cc=mpicc --with-xbraid-lib=/home/sguenther/Software/braid_warm_restart_export/ --with-xbraid-include=/home/sguenther/Software/braid_warm_restart_export --enable-autodiff
// erstellt SU2_BASE und SU2_AD Ordner und macht dann configure in beiden:


>make -j install 
%compiliert beides, wenn es im Hauptordner ausgeführt wird -> 2 binaries: su2_CFD & su2_CFD_AD für normale und die Codi Variante



RUN:
==========
in Testcases/LaminarAirfoil...:
primal:
>mpirun -np 4 ../../bin/SU2_CFD laminar_airfoil_usw.cfg

adjoint: (modify config file for adjoint
>mpirun -np 4 ../../bin/SU2_CFD_AD laminar_airfoil_usw.cfg
